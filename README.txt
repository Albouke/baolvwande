# 项目介绍
    本项目是针对小袁口算 安卓端 3.93.2版本 “口算大挑战”webview页面的修改程序，能够自动抓包并修改题目对错的布尔判断函数
# 环境依赖
mitmproxy及对应的证书文件
# 目录结构描述
    ├── README.txt          // 帮助文档
    
    ├── autoRestart.py    // 启动端口pid自动清除及启动挂载了自动覆写脚本的监听器的脚本
    
    ├── finderGUI.py      // 一个用于查询本机ip状态的工具脚本，用于查询代理ip
    
    └── replace.py           //挂载于mitmproxy的自动覆写脚本
 
# 使用说明
请使用系统应用-设置-WLAN-修改网络-代理（手动）设置 代理服务器 程序默认监听7778端口，ip请使用工具或自行查询
预先安装mitmproxy监听对应端口并导入证书至宿主机及被监听机，确保mitmproxy权限，请自行查询mitmproxy的下载及证书配置
请使用系统应用-设置-应用-小袁口算-储存空间-清除缓存，确保监听时重载webview页面
使用autoRestart.py脚本启动监听服务器
